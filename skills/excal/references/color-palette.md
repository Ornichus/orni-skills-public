# Color Palette

Single source of truth for all colors used in Excalidraw diagrams generated by this skill.

---

## Shape Colors (Semantic)

| Purpose | Fill | Stroke | When to Use |
|---------|------|--------|-------------|
| Primary/Neutral | `#3b82f6` | `#1e3a5f` | Main flow elements, default |
| Secondary | `#60a5fa` | `#1e3a5f` | Supporting elements |
| Tertiary | `#93c5fd` | `#1e3a5f` | Background/context elements |
| Start/Trigger | `#fed7aa` | `#c2410c` | Entry points, triggers, inputs |
| End/Success | `#a7f3d0` | `#047857` | Completion, success states |
| Warning/Reset | `#fee2e2` | `#dc2626` | Warnings, resets, destructive |
| Decision | `#fef3c7` | `#b45309` | Conditionals, branches |
| AI/LLM | `#ddd6fe` | `#6d28d9` | AI components, model calls |
| Inactive/Disabled | `#dbeafe` | `#1e40af` | Disabled states (use dashed stroke) |
| Error | `#fecaca` | `#b91c1c` | Error states, failures |

**Rule:** Always pair darker stroke with lighter fill.

---

## Text Colors (Hierarchy)

| Level | Color | Use |
|-------|-------|-----|
| Title | `#1e40af` | Section headings, major labels |
| Subtitle | `#3b82f6` | Subheadings |
| Body/Detail | `#64748b` | Descriptions, annotations |
| On light fills | `#374151` | Text inside light-colored shapes |
| On dark fills | `#ffffff` | Text inside dark-colored shapes |

---

## Evidence Artifact Colors

| Artifact | Background | Text Color |
|----------|------------|------------|
| Code snippet | `#1e293b` | Syntax-colored (language appropriate) |
| JSON/data | `#1e293b` | `#22c55e` (green) |

---

## Arrow & Line Colors

- **Arrows:** Use source element's stroke color
- **Structural lines** (dividers, trees, timelines): `#1e3a5f` or `#64748b`
- **Marker dots:** `#3b82f6`

---

## Background

- **Canvas:** `#ffffff`
