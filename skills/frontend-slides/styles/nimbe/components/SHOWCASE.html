Large Language Models

Les LLM sont des réseaux neuronaux entraînés sur d'immenses corpus textuels pour modéliser la distribution probabiliste du langage. Ils permettent génération, traduction, résumé, raisonnement.

Acteurs majeurs

  • OpenAI — GPT-4o, o1 (raisonnement)
  • Anthropic — Claude 4.7 Opus / Sonnet
  • Google DeepMind — Gemini 2 Pro
  • Mistral AI — modèles open-weight souverains

Usage type

Les LLM motorisent les agents Claude Code utilisés en interne pour automatiser le développement, la documentation, les présentations, et l'orchestration multi-projets.

Calcul quantique

L'informatique quantique exploite les phénomènes de superposition et d'intrication pour résoudre certaines classes de problèmes en temps polynomial là où les machines classiques sont exponentielles.

Algorithmes clés

  • Shor — factorisation entière (casse RSA)
  • Grover — recherche non structurée (accélération √N)
  • VQE / QAOA — optimisation hybride classique-quantique

Horizon

Systèmes commerciaux fault-tolerant attendus 2027-2030. Risque "Harvest Now, Decrypt Later" déjà actif sur les archives chiffrées RSA/ECC sensibles.

Fusion nucléaire

Reproduction sur Terre des réactions stellaires : fusionner deutérium et tritium pour libérer une énergie 4× supérieure à la fission, sans déchets longue durée ni risque d'emballement.

Acteurs

  • Commonwealth Fusion Systems — SPARC tokamak compact 2026
  • Helion Energy — pulse fusion, contrat Microsoft 2028
  • TAE Technologies — boron-11 aneutronique
  • ITER — international, première plasma deutérium 2035

Implication IA

L'IA est aujourd'hui contrainte par l'énergie disponible (~5% de la conso US prévue pour les datacenters d'ici 2030). Énergie quasi-illimitée = modèles quasi-illimités.
